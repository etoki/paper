# ハラスメント・シミュレーション拡張：好奇心ベースの分析記録

> 本ドキュメントは `simulation/curiosity_scarcity/` で行った 4 段階の探索的分析の総括。
> 本論文（OSF DOI 10.17605/OSF.IO/3Y54U、v2.0 master）には反映していない。
> 完全な好奇心ベースの拡張で、pre-registration 範囲外の post-hoc 分析。

---

## 出発点となった問い

> 性格が悪くても制度が良ければ悪い行為は起こらず、良い性格でも制度が悪ければ悪い行為が促進される — これが本論文の結論。
> ただし厳罰化（制度介入）しても犯罪が無くならないこともある。資源不足・技術不足・生産性低下が悪い行為を後押しする可能性。
> ハラスメント文脈では具体的にどう現れ、どこまでシミュレーションに落とし込めるか？

この問いから出発し、4 つのバージョンで段階的に精緻化した。

---

## バージョン履歴と主な発見

### v1: 仮の数字での試作

ハラスメント発生率を上下に動かす **3 つの力** を初期実装：

- **制度** （下げる）：上から課されるルール
- **資源不足** （上げる）：人手不足・過重労働
- **自制心不足** （上げる）：個人の trait

仮の係数（γ_D = 0.5、γ_E = 0.4）で計算。
**結論: 「制度は人手不足 50% で簡単に打ち消される」** という強い主張が出た。

→ ただし係数の根拠が弱く、Bowling & Beehr 2006 の **被害者視点** の数字を借りていた。
本来の問い（加害者の行動）には合わない可能性があった。

### v2: 文献 5 本で正しく校正

ユーザーが追加で取得した PDF 5 本から、**加害者側のメタ解析** で校正し直した：

| パラメータ | v1 値 | v2 値 | 出典 |
|---|---|---|---|
| 資源不足の係数 γ_D | 0.50 | **0.30** | Hershcovis 2007（加害者側メタ）|
| 自制心不足の係数 γ_E | 0.40 | **0.22** | De Ridder 2012（自制心メタ）|

加えて 3 種類の数式（identity / log-linear / logit）の感度比較、信頼区間の sweep、
共通方法分散補正の sweep を追加。

**結論の逆転**：v2 校正後では「制度は中程度の人手不足には勝つ」。
v1 で出た「簡単に負ける」は **被害者側の数字を加害者側に流用したことによる過大評価** だった。

### v3: 3 つの追加診断

#### 診断 1：実データでクラスタと発生率を見る

7 つの性格クラスタ別に、自制心の値と実際のハラスメント発生率を比較：

```
クラスタ  自制心  発生率
C5        2.27    3.8%   ← 最低（自制心も最低なのに）
C0        2.89   39.1%   ← 最高（自制心は中くらい）
C3        3.94    8.2%
```

クラスタ平均の相関係数 = +0.034（ほぼ無相関）。

**「自制心が低い人ほどハラスメントが多い」というよくある仮説は、本データでは成立しない**。
代わりに目立つのは C0 の特徴：「社交的（高 X）× 利己的（低 HH）」の組み合わせ。
v2 で組み込んだクラスタごとの増幅は撤回すべきと判明。

#### 診断 2：竜巻図（どの仮定が結論を最も動かすか）

11 個の仮定を一つずつ振って影響を測定：

```
1. 制度の強さ effect_C 0.10 ↔ 0.30           ★★★★ ← 最大
2. 資源不足の係数 γ_D の不確実性             ★★
3. その他全部                                ★ 以下
```

**結論を一番動かすのは「制度の強さ」**。γ や数式選択などの細かい仮定は影響が小さい。

#### 診断 3：時間軸の感度

4 つの時間 regime で資源不足の係数を scaling：

| Regime | 倍率 | 解釈 |
|---|---|---|
| 急性（1 日 spike）| × 0.5 | 単発、影響弱い |
| 横断的（基準）| × 1.0 | 多くの研究の design |
| 縦断的（De Ridder データ）| × 0.61 | 時間が経つと影響が薄れる |
| Hobfoll COR 理論 | × 1.4 | 慢性ストレスの spiral |

**経験データ（De Ridder）と理論（Hobfoll）が真逆を予測**。
今ある PDF だけでは決着できない。長期 panel データが必要。

### v4: 4 軸モデル（最終）

ユーザーの指摘で「技術」の定義を整理：
- 通報窓口・HR は **制度** 側に分類
- 監視カメラ・AI 検知も **制度** 側
- **技術 = 加害者本人が能動的に使う自己抑制ツール**（怒り管理アプリ、CBT など）

これにより 4 軸モデルが完成：

| 軸 | 何 | 主体 |
|---|---|---|
| **制度 C** | 上から課されるルールと監視 | 組織・社会 |
| **道具 T** | 加害者が自発的に使う抑制ツール | 加害者本人 |
| **資源不足 D** | 組織レベルの stressor | 環境 |
| **能力不足 A** | 個人 trait の自制心欠如 | 素質 |

**式**：
```
発生率 = 元の値
       × (1 - 制度) × (1 - 道具の効き目 × 利用率)
       × (1 + γ_D × 資源不足) × (1 + γ_A × 能力不足)
```

道具側の係数：
- 道具の効き目 T = 0.25（Hudson 2023 が唯一の anchor、harassment outcome 直接測定なし）
- 利用率 uptake = 0.20（健康プログラムからの借用、ハラスメント文脈では anchor なし）
- 道具の実効力 = T × uptake = **5%**（制度 20% の 4 分の 1）

## 最終的な強さ比較

各力の最大値を並べると：

| 力 | 最大値 | 出典 |
|---|---|---|
| 資源不足 | **+30%** | Hershcovis 2007 加害者メタ |
| 能力不足 | +22% | De Ridder 2012 自制心メタ |
| 制度 | **-20%** | 4 種類の研究の三角測量 |
| 道具（実効）| -5% | Hudson 2023 |

→ **絶対値で比較すると、資源不足のほうが制度より強い**。

### 制度はいつ負けるか

制度と資源不足だけで考えると、

```
0.80 × (1 + 0.30 × 資源不足) = 1.0
資源不足 = 0.83
```

**人手不足が 83% を超える職場では、制度を強化しても発生率は元の値を超える**。

### 場面別の結果

元の発生率 17.2% に対して：

| 状況 | 発生率 | 制度の勝敗 |
|---|---|---|
| 余裕のある職場 | 13.8% | 完勝 ✅ |
| やや忙しい（資源不足 30%）| 15.0% | 勝ち ✅ |
| 慢性的不足（資源不足 50%）| 15.8% | 勝ち ✅ |
| 限界（資源不足 83%）| 17.2% | 引き分け ⚖️ |
| 崩壊（資源不足 100%）| 17.9% | 負け ❌ |

**資源不足 + 自制心不足の両方が中程度（各 50%）だと制度は負ける**。

## 「道具で能力を補える」の判定

ユーザーの問題提起「能力が低くても道具で代替できることもある」について：

| 視点 | 結論 |
|---|---|
| 理論的には正しいか？ | ✅ 正しい — 余裕のある職場では能力 trait の影響を打ち消せる |
| 量的に意味があるか？ | ⚠️ 小さい — 制度の 4 分の 1 程度 |
| 現実的に機能するか？ | ❌ 怪しい — 高ストレス職場では利用率が下がって機能しない |

**最も道具が必要な「資源不足の職場」では、道具の補完機能が崩壊する**という皮肉な構造が見えた。

理由：
1. 資源不足が直接ハラスメントを増やす
2. 道具を使う心の余裕がなくなり、利用率も下がる
3. 制度・道具・能力がすべて機能不全に陥る

---

## 政策的含意

優先度の高い順：

1. **資源不足を改善する** — 全部の前提（人員配置、労働時間規制）
2. **制度を強化する** — 一番効く（最大 20% 削減）
3. **道具の利用率を上げる仕組み** — 補助（強制研修、有給での参加）
4. **個人の能力訓練** — 一番効きにくい（trait は変えにくい）

❌ **「ハラスメント防止アプリを配って終わり」はほぼ効かない**。
資源不足と制度を放置したまま個人技に頼っても無効。

## 残った正直な限界

| Issue | 内容 |
|---|---|
| 加害者側 vs 被害者側のメタ解析の差 | Hershcovis (.30) vs Bowling & Beehr (.53)。本当の生成効果は中間にある可能性 |
| 自制心メタも自己報告同士 | 共通方法分散で過大評価の可能性。直接測定の longitudinal panel が必要 |
| 道具側の anchor が弱い | Hudson 2023 一本のみ、harassment outcome 直接測定なし |
| 利用率の anchor 不在 | 健康分野からの借用、ハラスメント防止アプリの literature は事実上存在しない |
| 資源不足が利用率を下げる係数 | 完全に理論からの推測、直接検証データなし |
| 数式の選択 | 文献では決定不能、3 種類の感度比較で透明化済み |
| 時間軸 | 経験と理論で逆方向、long panel が必要 |
| モデル全体は post-hoc | pre-registration 範囲外、modeling choice 含む |

これらは **追加の文献調査では解決しない**。ハラスメント防止技術の系統的臨床試験や、
労働者パネル調査（J-MILLS など）が出るまでの placeholder として残す。

## 結論を 3 行で

> 1. **制度（厳罰化）は中程度の資源不足には勝つが、絶対値では資源不足のほうが強い**
> 2. **資源不足が極端な職場では、制度を強化しても基準値を超えるハラスメントが起こる**
> 3. **「道具で能力を補える」は理論的には正しいが、最も必要な高ストレス職場では機能しない**
