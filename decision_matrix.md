## Scoring rubric (per venue)

Each venue is scored 1 (poor) to 5 (excellent) on six axes; total is the
weighted sum. Weights reflect this project's specific situation
(deadline pressure, single author, preprint already public).

| Axis | Weight | Definition |
|------|-------:|-----------|
| **Acceptance fit** | 0.25 | How well the venue's scope matches a Big Five x online-learning meta-analysis. |
| **Indexing quality** | 0.20 | Scopus / WoS / IEEE Xplore presence — what SFC actually counts. |
| **Deadline feasibility** | 0.20 | Can the novel analysis + paper realistically be done before the deadline given competing venues? |
| **Self-plagiarism risk** | 0.15 | How cleanly the novel analysis can be carved out from the preprint without text overlap. |
| **Travel / cost** | 0.10 | Total cost (registration + travel + lodging) in USD-equivalent for a Japan-based author. |
| **Strategic value** | 0.10 | Does this venue open a community / journal pipeline beyond the immediate point? |

Total = 5.0 max. Cutoff for "submit": ≥ 3.5.

These have NOT been verified in the same depth; if needed, a fact-finding
sweep is required before relying on them. Listed by the calendar window
they would plausibly fill:

- **EARLI 2027** (biennial; would not help 2026 deadline).
- **AERA 2027 Annual Meeting** — abstract-only, no proceedings; doesn't count as publication for SFC.
- **EDULEARN 2026** (IATED, July) — ICERI sister event, similar indexing.

If a Tier 1 paper is rejected, draft a triage note:
1. Which novel analysis was the spine of the rejected paper?
2. Which Tier 2 venue absorbs that analysis without retooling?
3. Is the rejection's reviewer feedback actionable for a Tier 2 resubmission, or does it require a return to the meta-analytic step?

## Why **not** these other paths

| Path | Why excluded |
|------|--------------|
| Direct journal submission (e.g., Computers & Education, BJET) | Multi-month review cycle; will not produce an acceptance by end of 2026. The journal version is on the post-2026 roadmap, with the preprint as its base. |
| Withdraw preprint and republish in a journal | Would break the "preprint maintained" commitment to the field; preprint already cited / indexed by Research Square (DOI assigned 2026-04-27). |
| Single conference, single submission | Rejection-risk concentrated in one venue; insufficient redundancy to meet the 2026 deadline. |
| Multi-author collaboration | No co-authors available; standing decision. |
