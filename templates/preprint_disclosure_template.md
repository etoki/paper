- [ ] preprint の DOI が正確: `10.21203/rs.3.rs-9513298/v1`
- [ ] Novel analysis が manuscript の実態と一致(誇張なし、欠落なし)

- [ ] 投稿者の prior IEEE / Frontiers 業績(P1, P2)が必要なら listing 済み

- [ ] ORCID `0009-0009-7124-6669` を含む
- [ ] 学会の cover letter テンプレートが別途ある場合はそれを優先
