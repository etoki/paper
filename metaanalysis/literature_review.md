# Big Five × Online Learning: Literature Review

本ドキュメントはメタ分析論文のための先行研究一覧である。

## スコープ（確定）

| 項目 | 確定内容 |
|------|---------|
| 主題 | Big Five 性格特性 × オンライン学習 |
| モダリティ | 完全オンライン / blended / MOOC すべて含む（moderator 化） |
| 主アウトカム | 学業成績（achievement: GPA, 成績, テストスコア等） |
| 副アウトカム | 満足度, エンゲージメント（データが集まれば） |
| 教育段階 | 全年齢（K-12, 大学, 大学院, 成人学習） |
| 性格フレームワーク | Big Five (BFI, NEO-PI, IPIP, BFI-2) + HEXACO（MBTI は除外） |
| 言語 | 英語のみ |
| 期間 | 制限なし（ただし 2003 以降が実質） |

## 含有／除外基準

### Inclusion
1. Big Five or HEXACO inventory で性格測定
2. オンライン学習環境（完全オンライン / blended / MOOC / LMS ベース）での学習
3. 学業成績 or 関連量的アウトカム報告
4. 英語論文
5. Peer-reviewed または preprint

### Exclusion
1. MBTI 等の type-based personality framework
2. 完全対面のみ（オンライン要素なし）
3. 量的効果量が抽出不能（質的研究のみ）
4. 非英語論文
5. 重複サンプル（同データが複数論文で報告）は一つに統合

## 検証状態の凡例
- ✓ **検証済**: DOI/Scholar/PubMed で書誌情報確認
- ⚠ **要最終確認**: 1 つ以上の情報が未確定（PDF で要確認）
- ❌ **要除外**: スコープ外確認済

## 統計記号
- *r*: ピアソン/スピアマン相関
- β: 標準化回帰係数
- d: Cohen's d
- N: サンプルサイズ
- "p<.05" 等は統計的有意水準
