# 移行完了後の各学会への開示文面

preprint migration 完了後、cover letter / submission portal 内で開示する文面集。

`metaanalysis/conference_submissions/templates/preprint_disclosure_template.md` の Version C をベースに、各学会向けに微調整。

## ECEL 2026 用

```
Preprint disclosure:
A preliminary version of this manuscript was previously hosted on Research Square
(DOI 10.21203/rs.3.rs-9513298/v1, posted 2026-04-27). The authors have since migrated
the work to OSF Preprints / PsyArXiv (DOI [NEW_DOI]) on [DATE], a not-for-profit
preprint server. The Research Square version has been formally withdrawn (or is in
the process of withdrawal). The submission to ECEL 2026 represents an extended
version with [list specific extensions, e.g., updated risk-of-bias analysis,
new sensitivity analyses, sole-authored discussion expansion].

The authors confirm that this work has not been published in, and is not under
review at, any peer-reviewed venue.
```

## IEEE TALE 2026 用 (P5 を投稿の場合)

```
Preprint disclosure (IEEE compliance):
A preliminary version of this work is hosted on OSF Preprints (DOI [P5_OSF_DOI])
and SocArXiv (DOI [P5_SOCARXIV_DOI]). Both servers are not-for-profit and
consistent with IEEE's preprint policy. Upon acceptance, the corresponding author
will submit the IEEE Copyright Form. No version of this work has been published
in or accepted by any other peer-reviewed venue.
```

## IEEE TALE 2026 用 (P3 を投稿する場合 — preprint移行が前提)

```
Preprint disclosure (IEEE compliance):
This manuscript was previously posted on Research Square. To comply with IEEE's
preprint policy, the work has been migrated to OSF Preprints (DOI [NEW_DOI]),
a not-for-profit server. The Research Square version has been withdrawn. No
version has been published in or accepted by any peer-reviewed venue.
```

## 重要な原則

1. **誠実な disclosure** が long-term で最も安全な戦略
2. preprint を隠して投稿 → 後で発覚すると academic misconduct 扱いされる可能性
3. 各学会の policy をよく読み、不明な場合は事前照会
4. 投稿時の cover letter / submission form の "any prior dissemination" 欄に必ず記載
