## 14. Review question

```
In online learning environments (including fully online, blended, MOOC,
and LMS-mediated learning), what is the pooled magnitude and direction of
the association between Big Five (or HEXACO) personality traits and
academic achievement? How does this association vary by learning modality,
education level, region, and era (pre/during/post COVID-19)?
```

## 18. Participants/population

### Inclusion criteria

```
Students of any educational level (K-12, undergraduate, graduate, adult learners)
enrolled in online learning environments. No geographic or demographic restrictions.
```

### Exclusion criteria

```
Fully face-to-face samples with no online component. Studies where learners
are not human (e.g., AI agents).
```

## 19. Intervention(s), exposure(s)

```
Big Five personality traits (Conscientiousness, Openness, Extraversion,
Agreeableness, Neuroticism) or HEXACO personality traits as measured by
validated inventories (BFI, BFI-2, NEO-PI-R, NEO-FFI, IPIP, HEXACO-PI-R,
or comparable).
```

## 20. Comparator(s)/control

```
Not applicable — review is of observational correlation studies.
No explicit comparator group required.
```

## 21. Types of study to be included

```
Included:
- Observational correlational studies (cross-sectional, longitudinal)
- Studies reporting sufficient statistics for effect size extraction

Excluded:
- Studies using MBTI or other non-Big Five/HEXACO personality frameworks
- Qualitative studies only
- Commentary, editorial, review papers (but their reference lists will be scanned)
- Case studies (N < 10)
```

## 22. Context

```
Online learning environments — including fully online courses, blended/hybrid
learning, MOOCs, and LMS-mediated learning. Synchronous and asynchronous modes.
Contexts include K-12, higher education, graduate, and adult learning worldwide.
```

## 23. Main outcome(s)

### Primary outcome

```
Academic achievement — operationalized as GPA, course grades, test scores,
learning performance, or equivalent quantitative academic outcomes.
```

### Measures of effect

```
Pearson correlation coefficient (r) as primary effect size, with 95% confidence
intervals. Fisher's z transformation used for pooling. Reporting also includes
sample-weighted means, heterogeneity (I², τ², Q), and prediction intervals.
```

## 24. Additional outcome(s)

```
Secondary outcomes (if data allow):
- Academic satisfaction
- Academic engagement
- Learning-related behaviors (e.g., LMS use, completion rates)
```

## 25. Data extraction (selection and coding)

```
One reviewer will extract data from included studies using a structured
form. A 10% random subsample will be re-extracted after a 1-week interval
to assess intra-rater reliability (Cohen's κ); κ ≥ 0.80 will be targeted.

Extracted fields:
- Study identification (author, year, DOI)
- Sample characteristics (N, age, gender, education level, country)
- Study design and learning context (modality, synchronous/asynchronous,
  duration, subject domain, platform)
- Personality measure (instrument, reliability)
- Outcome measure (instrument, reliability)
- Effect sizes (r, β, or convertible statistics)
- Risk of bias assessment

Data will be managed in a structured spreadsheet (CSV).
```

## 26. Risk of bias (quality) assessment

```
Risk of bias will be assessed using the Joanna Briggs Institute (JBI)
critical appraisal checklist for analytical cross-sectional studies,
across 8 domains:
1. Sampling frame appropriateness
2. Participant/setting description
3. Personality measure validity/reliability
4. Sample size justification
5. Outcome measure objectivity
6. Statistical analysis appropriateness
7. Confounder identification and adjustment
8. Ethics reporting

Each item will be rated Yes / No / Unclear. An aggregate score will be
used as a moderator in sensitivity analyses.
```

## 27. Strategy for data synthesis

```
Random-effects meta-analysis using restricted maximum likelihood (REML)
estimator. Hartung-Knapp-Sidik-Jonkman (HKSJ) adjustment for confidence
intervals. Effect sizes (Pearson r) converted to Fisher's z for pooling,
back-transformed for reporting.

Five primary analyses (one per Big Five trait):
1. Conscientiousness × achievement
2. Openness × achievement
3. Extraversion × achievement
4. Agreeableness × achievement
5. Neuroticism × achievement

Heterogeneity assessed via Q, I², τ², and 95% prediction intervals.
Forest plots and funnel plots produced for each trait.

HEXACO studies will be mapped to Big Five according to Ashton & Lee (2007):
H-Emotionality ≈ Neuroticism (sensitivity analysis for mapping robustness).

Analysis software: R (metafor package, Viechtbauer 2010).
```

## 28. Analysis of subgroups or subsets

```
Planned subgroup and meta-regression analyses (moderators):
1. Modality: fully online / blended / MOOC / synchronous / asynchronous
2. Education level: K-12 / undergraduate / graduate / adult
3. Region: Asia / Europe / North America / Other
4. Era: pre-COVID (≤ 2019) / COVID (2020–2022) / post-COVID (2023–)
5. Outcome type: GPA / exam / LMS behavior / composite
6. Personality instrument: BFI / BFI-2 / NEO-FFI / NEO-PI-R / IPIP / HEXACO
7. Publication year (continuous)
8. Sample size (log-transformed, continuous)
9. Risk of bias score (continuous)

Multiple comparison correction using Holm-Bonferroni method.
```
